# 数组扁平化，去重，排序

var arr = [ [1, 2, 2], [3, 4, 5, 5], [6, 7, 8, 9, [11, 12, [12, 13, [14] ] ] ], 10];

**扁平化**

* reduce 写法

```js
function flat (arr) {
	return arr.reduce((cur, next) => {
		if (Array.isArray(next)) {
			return [...cur, ...flat(next)]
    	} else {
			return [...cur, next]
        }
	}, [])
}
```

* 普通写法

```js
function flat () {
  return [].concat(...this.map(item => (Array.isArray(item) ? item.flat() : [item])));
}
```

**去重**

```js
[...new Set(flat(arr))]
```
