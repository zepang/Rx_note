### 第一章 作用域是什么？

存储和查找变量的规则。

1.1 编译原理：
* JavaScript实际上一门编译语言。任何JavaScript的代码在执行前都要进行编译。大部分的编译发生在代码执行前的几微妙（甚至更短）。
* 编译的三个过程：
  * 分词/词法分析
  * 解析/语法分析
  * 代码生成
1.2 作用域的理解
* 对 JavaScript 代码进行处理时需要参与的三个成员 引擎，编译器，作用域以及它们分别的作用。
* 作用域参与编译器和引擎的工作，协助编译器和引擎存储和查找变量
